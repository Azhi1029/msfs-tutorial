PHASE 01

TACAN 基礎概念

🔵
TACAN 是什麼？ TACAN（Tactical Air Navigation）是軍用無線電導航系統，功能類似民用 VOR/DME。F/A-18C 使用 TACAN 獲得：
· 方位角（Bearing）：飛機相對於 TACAN 站的方向
· 距離（DME）：飛機到 TACAN 站的距離（海浬）
TACAN 可指向地面站、航母、僚機（A/A 模式），是 DCS 任務最常用的導航工具。
💡
TACAN 頻道說明
頻道範圍用途例子
1–126 X/Y地面站 / 機場26X（貝魯特）
1–99 X/Y航母（加 63）71X（CVN-74）
1–63 X/Y空對空（A/A 模式）1X（僚機）
X 和 Y 是頻道後綴，任務簡報會指定。最常用的是 X 頻道。
完成進度
0%
1
了解 TACAN 的三種模式
· T/R（Transmit/Receive）：最常用，收發兼具，獲得方位角和距離
· RCV（Receive Only）：只接收，不發送，匿蹤時使用（只有距離，無方位）
· A/A T/R：空對空模式，追蹤僚機或加油機位置
✅
2
了解 HSI 上的 TACAN 資訊顯示
DDI 的 HSI 頁面顯示：
· TACAN 針（菱形）：指向 TACAN 站的方向
· BRG 數值：方位角（如 BRG 045 = 東北方）
· 距離數值：到 TACAN 站的距離（如 28NM）
· HUD 右下角也會顯示 TACAN 距離
✅
3
了解 Bearing vs Radial 的差異
· Bearing（方位）：從我飛機到 TACAN 站的方向（HSI 上的針）
· Radial（輻射線）：從 TACAN 站向外的方向（ATC 指定進場方向用）
例：在站的正北方，Bearing = 180°（朝南飛向站），Radial = 360°（在站的 360 輻射線上）
✅
🎛️
PHASE 02

TACAN 面板設定

✅
設定位置 TACAN 面板在 右側主控台（Right Console）後方區域。UFC（上前方控制面板）也可輸入 TACAN 頻道。兩種方法都會教。
完成進度
0%
方法一：TACAN 實體面板
1
右主控台 → 找到 TACAN 面板
TACAN 面板在右側主控台後方。有兩個旋鈕（X10 和 X1 數字旋鈕）、X/Y 切換、模式選擇開關（OFF / RCV / T/R / A/A T/R）。
✅
2
旋鈕設定頻道數字（如 071）
左旋鈕控制 X10 位數（0–12），右旋鈕控制 X1 位數（0–9）。例如設定 071：左旋鈕 → 07，右旋鈕 → 1。DCS 中用滑鼠滾輪或左右點擊旋鈕。
✅
3
X/Y 切換 → 設定為 X（通常）
TACAN 面板上的 X/Y 切換開關，一般任務使用 X 頻道。任務簡報若指定 Y 則切換。
✅
4
模式開關 → T/R（一般飛行）
模式開關從 OFF 轉至 T/R。幾秒後 DDI HSI 頁面的 TACAN 針開始作動，顯示方位和距離。若要追蹤僚機，改用 A/A T/R。
✅
方法二：UFC 快速輸入
5
UFC → 按 TCN 按鈕
UFC（上前方控制面板，HUD 下方）按下 TCN 按鈕，顯示器出現 TACAN 輸入提示。
✅
6
鍵入頻道數字 → 按 ENT 確認
用 UFC 數字鍵輸入 071（或任務指定頻道），再按 ENT 確認。UFC 顯示器顯示設定的頻道，TACAN 自動切換。
✅
7
確認 DDI HSI 頁面 TACAN 針作動
DDI → HSI 頁面，確認 TACAN 菱形針出現並指向正確方向，底部顯示 BRG xxx / xxNM。若針不動，確認 TACAN 模式在 T/R 且頻道正確。
✅
🗺️
PHASE 03

HSI 判讀與 TACAN 導航

🔵
HSI = Horizontal Situation Indicator（水平情境顯示器） HSI 是 F/A-18C 最重要的導航頁面，整合 TACAN、航路點、進場資訊。學會看 HSI 是所有導航的基礎。
完成進度
0%
1
DDI → MENU → HSI 頁面開啟
右 DDI 按 MENU，再按 HSI（或 NAV）。HSI 頁面顯示：中央羅盤、飛機符號（中心）、TACAN 菱形針、航路點指針、課程線（CRS）。
✅
2
讀取 TACAN 方位角（Bearing）
HSI 上的 菱形針（◆）尖端指向 TACAN 站方向。
· 針尖在正上方 = TACAN 站在正前方
· 針尖在右方 = TACAN 站在右側
底部顯示 BRG xxx°（絕對方位角）和距離 xx.xNM。
✅
3
Course Line（CRS）設定 — 對準航道
HSI 上的 Course Line 是你想飛的航道方向。用 DDI 左側的 OSB 按鈕或 UFC 調整 CRS 數值。CRS 線對準 TACAN 菱形針 = 你正沿著最短路徑飛向 TACAN 站。
✅
4
HUD 上的 TACAN 資訊判讀
HUD 右下方顯示 TACAN 距離（如 28.3）。HUD 左下方顯示目前導航模式。注意：HUD 顯示的是距離，不顯示方位，需配合 HSI 判斷方向！
✅
5
實際導航：飛向 TACAN 站
操作步驟：
1. 看 HSI 確認 TACAN 針方向
2. 轉向讓 TACAN 針指向正上方（= 飛機朝向 TACAN 站）
3. 保持 TACAN 針在正上方飛行
4. 距離持續減少 → 表示正確飛向 TACAN 站
5. 距離接近 0 → 飛越 TACAN 站上空
✅
6
Offset 導航：不直飛 TACAN 站
若不想直飛 TACAN 站（如 TACAN 在敵區），可設定 CRS 偏移飛平行航道。例如 TACAN 在東方 BRG 090°，想在其北方 5NM 飛過，則設 CRS = 090 + 偏移角，在適當距離時轉向。
✅
📍
PHASE 04

航路點輸入

🔵
F/A-18C 航路點系統 F/A-18C 最多可儲存 60 個航路點（Waypoint），分為：
· WP 1–25：一般航路點（任務前在 DCS 任務編輯器或飛行中輸入）
· WP 26–29：Mark Points（飛行中標記地面目標用）
· 建議：在任務編輯器預先設好航路點，飛行中用 DDI 選擇即可
完成進度
0%
方法一：DDI 選擇現有航路點
1
DDI → MENU → DATA → WYPT
右 DDI：MENU → DATA → WYPT，顯示航路點列表。任務編輯器預設的航路點已在此清單，可查看每個點的座標、高度和距離。
✅
2
選擇目標航路點（按 OSB 選擇編號）
WYPT 頁面用 OSB（屏幕邊按鈕）選擇想要前往的航路點編號。選中後按 UFC → WYPT 或 DDI 的 SEQ 按鈕啟動序列。HSI 會自動顯示該航路點的指針。
✅
3
HSI 確認：航路點圓圈指針出現
HSI 頁面出現 圓圈指針（○）指向選定的航路點（有別於 TACAN 的菱形針）。底部顯示 WP X · xx.xNM。HUD 也會顯示航路點方向標誌。
✅
方法二：飛行中手動輸入座標
4
DDI → DATA → WYPT → 選空白點位
在 WYPT 頁面選擇一個空白的航路點編號（如 WP5），按 UFC 進入輸入模式。UFC 顯示器出現座標輸入提示：先輸入緯度（LAT）。
✅
5
輸入緯度（LAT）→ ENT
輸入緯度格式：N/S + 度 + 分（小數）
例：N25042 = 北緯 25° 04.2'
· 先按 N（或 S）
· 再輸入數字（如 25042）
· 按 ENT 確認緯度
✅
6
輸入經度（LON）→ ENT
接著輸入經度格式：E/W + 度 + 分（小數）
例：E121335 = 東經 121° 33.5'
· 先按 E（或 W）
· 再輸入數字（如 121335）
· 按 ENT 確認，航路點儲存完成
✅
7
設定航路點高度（選擇性）
WYPT 頁面可進一步設定該點的 目標高度（Altitude）。輸入高度（英尺），FMC 會計算到達該點的 TOD（頂端下降點）。一般飛行可省略。
✅
Mark Point（空中標記地面目標）
8
飛越目標上空時按 UFC → MARK
飛越目標時，按 UFC 的 MARK 鈕，飛機當前位置自動儲存為 Mark Point（WP 26–29）。可用於快速標記地面目標的大概位置，供後續武器攻擊參考。
✅

PHASE 05

序列飛行實作（完整任務導航）

🎯
實際任務導航流程 把前面學的整合起來，完整飛一段任務導航：起飛基地 → 航路點 1（Initial Point）→ 航路點 2（目標區）→ 航路點 3（撤退點）→ TACAN 回基地。
完成進度
0%
1
起飛前：TACAN 設基地頻道，確認 HSI 正常
起飛前設好 TACAN 頻道（基地或航母）。確認 HSI 的 TACAN 針有作動、顯示距離。這樣萬一迷航，隨時可以用 TACAN 找回基地方向。
✅
2
DDI → DATA → WYPT → 確認航路點清單
起飛後爬升至巡航高度途中，確認 WYPT 清單各點正確：
· WP1：Initial Point（IP）
· WP2：目標區附近
· WP3：Egress 撤退點
選擇 WP1 為目前導航目標。
✅
3
HSI 確認航路點指針，轉向飛往 WP1
HSI 顯示 WP1 圓圈指針，轉向讓指針朝正上方（飛機朝向 WP1）。HUD 顯示 ▶ WP1 · xxNM。保持高度巡航飛往 WP1。
✅
4
接近 WP1 → 切換至 WP2
距離 WP1 約 2–5 NM 時，UFC 按 WYPT ↑（或 DDI 按 SEQ）切換到下一個航路點 WP2。若有設定 自動序列（AUTO SEQ），飛越 WP1 後自動切換。
✅
5
目標區附近：切換 TACAN 到目標區地面站（若有）
若任務區有 TACAN 地面站，可切換頻道更精確定位。注意：接近目標區時考慮切換到 RCV（只接收）模式，減少發射訊號被敵方偵測的機會。
✅
6
撤退後：切換回基地 TACAN，導航回家
完成任務撤退後，UFC 切換 TACAN 回基地或航母頻道，HSI 的 TACAN 針指向基地方向，順著 TACAN 針飛回。距離持續縮小 = 正確回航。
✅
7
進場前：確認 TACAN 距離 & 高度，準備 CASE I/II/III
接近基地時，TACAN 距離降至 15–20 NM，開始下降。依天氣決定 CASE I（目視）、II（混合）或 III（儀器）進場程序，切換回進場教學頁面的步驟。
✅
