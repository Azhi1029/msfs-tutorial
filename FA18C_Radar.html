PHASE 01

雷達基礎概念

🔴
F/A-18C APG-73 雷達簡介 F/A-18C 裝備 AN/APG-73 脈衝都卜勒雷達，可在 ±70° 方位角、±60° 仰角範圍內搜索目標。三種主要空對空模式：
· RWS（Range While Search）：搜索模式，找目標
· TWS（Track While Scan）：同時追蹤多目標
· STT（Single Target Track）：鎖定單一目標，提供武器瞄準
完成進度
0%
1
了解雷達螢幕顯示
DDI 雷達頁面由上到下代表近到遠（B-Scope 顯示）：
· 螢幕上方 = 遠距離（如 40NM）
· 螢幕下方 = 近距離（如 0NM）
· 螢幕左右 = 方位角（左右各 ±20° 或 ±40°）
· 亮點 = 偵測到的目標回波
✅
2
了解關鍵雷達術語
· Vc（Closure Rate）：接近速率，負值（-）= 目標靠近，正值（+）= 目標遠離
· TA（Target Aspect）：目標機頭相對你的角度，0° = 目標朝你飛來（熱機頭），180° = 目標背對你
· ATA（Antenna Train Angle）：雷達天線相對機頭的偏轉角
· Notch：目標以 90° 角飛行使都卜勒雷達失去追蹤
✅
3
了解雷達掃描參數
· Range（距離）：搜索範圍，可選 10 / 20 / 40 / 80 / 160 NM
· Azimuth（方位角）：掃描寬度，±10° / ±20° / ±40° / ±60°
· Bar（掃描層數）：上下掃描幾層，1B / 2B / 4B（層數越多覆蓋越廣但更新越慢）
· 距離越遠、方位角越寬、Bar 越多 → 更新率越慢
✅
⚡
PHASE 02

雷達開機 & 基本設定

🔵
雷達開機位置 F/A-18C 雷達控制在 左側主控台（Left Console）的 RADAR 面板，以及 DDI 的雷達頁面（用 OSB 按鈕控制各參數）。引擎啟動後雷達需要約 3–4 分鐘暖機，起飛前務必提早開機！
完成進度
0%
1
左主控台 → RADAR 面板 → RADAR SW → OPR
左側主控台 RADAR 面板，將雷達開關從 OFF 撥至 OPR（Operate）。雷達開始暖機，DDI 顯示 STBY（待機） 約 3–4 分鐘後切換為 OPR。
✅
2
左 DDI → MENU → RADAR 頁面
左 DDI 按 MENU，再按 RADAR，進入雷達主頁面。螢幕中央是雷達顯示區（B-Scope），四周 OSB 按鈕控制各參數。確認右上角顯示 RWS 模式。
✅
3
設定搜索距離（RNG）→ 40NM（初始建議）
DDI 左側 OSB 按 RNG，調整搜索距離。初始建議設為 40NM（平衡覆蓋和更新率）。近距離纏鬥改為 10–20NM，長程 BVR 改為 80–160NM。
✅
4
設定方位角（AZ）→ ±40°（初始建議）
DDI OSB 按 AZ，切換方位角掃描寬度。±40° 是一般空戰的均衡設定。若已知敵機大概方向，縮小至 ±20° 加快更新率。
✅
5
設定掃描層數（BAR）→ 4B（初始建議）
DDI OSB 按 BAR，切換掃描層數。4B（4層） 覆蓋較廣的仰角範圍，適合搜索。鎖定目標後可改 2B 加快更新。
✅
🔍
PHASE 03

RWS — 搜索模式

📡
RWS（Range While Search）特性 RWS 是 最基本的搜索模式，雷達掃描整個搜索區域找目標。回波顯示為亮點，但不持續追蹤（每次掃描才更新一次位置）。偵測到目標後可按 TDC（目標標定控制）游標移到目標上，進入 TWS 或直接 STT 鎖定。
完成進度
0%
1
確認 DDI 雷達頁顯示 RWS 模式
DDI 雷達頁右上角顯示 RWS。若不是，按 DDI 右側 OSB 的 MODE 按鈕切換回 RWS。螢幕開始掃描，偵測到的目標顯示為亮點。
✅
2
識別目標回波：位置、距離、方位
看到亮點時判讀：
· 左右位置 = 方位角（中間 = 正前方）
· 上下位置 = 距離（上方 = 遠，下方 = 近）
· 回波旁的小數字 = 高度差（ft，正 = 目標在你上方）
RWS 只顯示位置，不顯示速度或航向！
✅
3
TDC（目標標定游標）移至目標上
用 HOTAS 上的 TDC（通常是右拇指搖桿） 將游標（十字線）移至目標回波上。游標對準目標後：
· 按壓 TDC = 進入 STT 鎖定（直接鎖定）
· 或先讓雷達自動切換 TWS 累積追蹤資料
✅
4
RWS 搜索技巧：調整天線仰角（EL）
若在 RWS 找不到目標，嘗試調整天線仰角：
· DDI OSB 按 CNTL 進入雷達控制頁
· 調整 EL（Elevation） 上下掃描角度
· 目標低空飛行時 EL 往下調，目標高空時往上調
· 也可用 HOTAS 天線仰角控制實時調整
✅
🎯
PHASE 04

TWS — 多目標追蹤模式

⚠️
TWS（Track While Scan）特性 TWS 在繼續搜索的同時持續追蹤最多 8 個目標。每個被追蹤的目標顯示為圓圈符號，並提供速度和航向資訊。TWS 是 AIM-120 AMRAAM 超視距作戰的核心模式，可同時對多個目標射擊。注意：TWS 不是完整鎖定，目標不一定知道被追蹤！
完成進度
0%
1
切換至 TWS 模式
DDI 雷達頁，按右側 OSB 的 TWS 按鈕，或按 MODE 切換。DDI 右上角顯示 TWS。雷達開始累積目標航跡資料，需要 2–3 次掃描才能建立追蹤（顯示圓圈）。
✅
2
識別 TWS 目標符號
TWS 頁面的目標符號：
· 空心圓圈（○）= 被追蹤的目標（System Track）
· 實心圓（●）+ 線= DT1（主要目標，Designated Target 1）
· 空心圓 + 線= DT2（次要目標）
· 旁邊的線條方向 = 目標飛行方向
· 數字 = 高度（千英尺）
✅
3
指定 DT1（主要目標）
TDC 游標移至想攻擊的目標圓圈上，按壓 TDC，該目標變為 DT1（Designated Target 1）。DT1 資訊出現在 DDI 右側：距離、方位、Vc（接近速率）、高度。HUD 出現目標符號和瞄準資訊。
✅
4
指定 DT2（次要目標）
TDC 移至第二個目標，按 TDC 側按（Castle Switch 短按） 指定為 DT2。同時有 DT1 和 DT2 時，可對兩個目標分別射擊 AIM-120（稍後武器教學詳述）。
✅
5
TWS 優缺點評估
✅ 優點：可追蹤多目標、目標 RWR 不一定有警告、仍可繼續搜索
❌ 缺點：追蹤精度較 STT 低（更新率慢）、目標做 Notch 動作易丟失
建議：BVR 遠距用 TWS，近距 10NM 內改 STT 確保鎖定精度
✅
🔒
PHASE 05

STT — 單目標鎖定模式

🔴
STT（Single Target Track）特性 STT 是完整鎖定，雷達停止搜索、全力追蹤單一目標，提供最精確的目標資訊。代價：目標的 RWR 會收到強烈警告訊號，知道被鎖定！STT 是 AIM-7 麻雀、Guns 機砲攻擊的必要條件，AIM-120 也可在 STT 模式下射擊。
完成進度
0%
1
從 RWS/TWS 進入 STT：TDC 對準目標按壓
RWS 或 TWS 模式下，TDC 游標移至目標回波/圓圈上，按壓 TDC（Depress），雷達鎖定目標進入 STT。DDI 右上角切換為 STT，目標符號變為十字準心（◎）。
✅
2
STT 鎖定後 DDI 資訊判讀
STT 鎖定後 DDI 顯示：
· 距離（RNG）：目標距離（NM）
· Vc（接近速率）：負值 = 目標靠近
· 方位（BRG）：目標方向
· 高度（ALT）：目標高度
· 目標機頭角（TA）：目標相對我的方向
✅
3
HUD 鎖定後顯示瞄準資訊
STT 鎖定後 HUD 顯示：
· 目標符號（◎）：目標在 HUD 視野內的位置
· ASE 圓（Allowable Steering Error）：瞄準允許誤差圈
· 距離條：左側縱線顯示目前距離與武器射程
· DLZ（Dynamic Launch Zone）：武器射程指示
✅
4
丟失鎖定：目標 Notch / 地雜波 / 干擾
以下情況會導致 STT 丟失鎖定（BREAK LOCK）：
· 目標 Notch（以 90° 橫切）：都卜勒效應消失
· 目標飛入地形遮蔽（地雜波）
· 目標投放箔條干擾（Chaff）
丟失後自動退回 RWS，需重新搜索鎖定。
✅
5
手動解除鎖定（Break Lock）
主動解除 STT 鎖定：按 Sensor Control Switch（SCS）後方 或 DDI 上的 BSGT（Break STT） 按鈕。解除後退回 RWS 繼續搜索，或切換至 TWS 追蹤多目標。
✅
⚔️
PHASE 06

實戰判讀技巧

🎯
雷達戰術整合 把前面的雷達知識整合成實戰流程：搜索 → 發現 → 識別 → 追蹤 → 鎖定 → 攻擊。每個步驟使用正確的雷達模式是存活的關鍵。
完成進度
0%
1
標準 BVR 接戰流程
1. RWS 搜索（80NM，4Bar）→ 發現目標
2. 目標進入 60NM → 切換 TWS，建立多目標追蹤
3. 指定 DT1 / DT2，40NM 射 AIM-120
4. 射擊後 Crank（轉向 ±60° 延遲 STT 鎖定）
5. AIM-120 Pitbull（自導）後可脫離或再攻
✅
2
目標機頭角（TA）判讀
· TA 0°（Hot）：目標正朝你飛來，Vc 最大（最危險！）
· TA 90°（Beam）：目標橫飛，Notch 可能，Vc 接近 0
· TA 180°（Cold）：目標背對你飛走，需追趕
目標 Hot（TA ≈ 0°）= 雙方都在靠近，AIM-120 最有效！
✅
3
Vc（接近速率）判讀與策略
· Vc -800 kt：雙方迎頭高速接近，射擊窗口短，需早射
· Vc -400 kt：一般接近速率，正常 BVR 接戰
· Vc 接近 0：目標可能在 Notch，考慮改變航向或切換模式
· Vc 正值：目標在逃跑，需加速或用 AMRAAM 追射
✅
4
雷達模式快速切換建議
距離建議模式原因
> 40NMRWS 80NM廣域搜索
20–40NMTWS多目標追蹤準備射擊
10–20NMSTT精確鎖定
< 10NMVACQ/GACQ近距纏鬥自動截獲
✅
5
常見雷達問題排解
問題原因解法
找不到目標仰角偏差調整 EL 上下掃描
鎖定一直丟失目標在 Notch轉向改變幾何關係
TWS 建不起來距離太遠 / 太近調整 RNG 參數
雷達不動作未暖機完成等待 STBY→OPR
✅
