AIM-120 AMRAAM

超視距主動雷達導引飛彈 · Fox 3 · TWS 多目標接戰

🚀
PHASE 01

AIM-120 AMRAAM 基礎

🟠
AIM-120 AMRAAM 簡介 AIM-120 AMRAAM（Advanced Medium-Range Air-to-Air Missile）是美軍主力超視距空對空飛彈，俗稱 「Slammer」。使用主動雷達導引，射出後飛彈自帶雷達尋找目標，不需要發射機持續鎖定（Fire and Forget）。射程依高度和速度從 20–80+ NM 不等。無線電呼號：FOX 3。
完成進度
0%
1
了解 AIM-120 飛行階段
· 慣性飛行（INS）：射出後依發射時的目標資料飛行，飛機可自由機動
· 資料鏈更新（Datalink）：若維持 TWS/STT，飛機持續傳送目標位置更新給飛彈
· 主動雷達（Pitbull）：飛彈啟動自身雷達自主追蹤，此後飛機可完全脫離
Pitbull 距目標約 10–15 NM 時啟動
✅
2
了解 DLZ（Dynamic Launch Zone）射程指示
HUD 左側的 DLZ 縱向條顯示目前射程區間：
· Rmax（綠色上方）：最大射程，射出後飛彈勉強到達
· Rne（黃色中間）：不逃脫射程，目標逃跑也能追到
· Rmin（紅色下方）：最小射程，太近射出飛彈無法引信
最佳射擊時機：游標進入 Rne 區間（黃色）
✅
3
AIM-120 各型號差異（DCS 中）
型號射程特點
AIM-120B~30NM（低空）早期型，DCS 常見
AIM-120C-5~40NM（低空）縮小彈翼，較常用
AIM-120C-7~50NM（低空）最新型，射程最遠
高空射擊射程可達 2–3 倍以上。高空、高速、迎頭 = 最大效益！
✅
⚙️
PHASE 02

武器設定與選擇

⚠️
射擊前必須確認 AIM-120 射擊前需要：MASTER ARM → ARM（主武器安全開關）、雷達在 TWS 或 STT 模式、目標在 DLZ 範圍內。MASTER ARM 在 左側主控台，ARM 位置才能實際射擊！
完成進度
0%
1
MASTER ARM → ARM（左主控台）
左側主控台，掀開護蓋將 MASTER ARM 開關撥至 ARM。HUD 右上角出現 ARM 字樣，武器系統啟動。訓練飛行保持 SAFE！
✅
2
DDI → SMS 頁面確認 AIM-120 掛載
右 DDI → MENU → SMS（Stores Management System），確認 AIM-120 掛載數量和位置（通常翼下和翼尖）。選擇武器模式為 A/A（空對空）。
✅
3
HOTAS 武器選擇 → AIM-120
用 Weapon Select Switch（WSS） 切換到 AIM-120 模式。HUD 左下角顯示 120C（或對應型號），DDI 雷達頁出現 DLZ 條。若有 AIM-9 和 AIM-120，WSS 上/下切換兩種飛彈。
✅
4
確認 HUD 顯示 AIM-120 射擊資訊
AIM-120 選定後 HUD 顯示：
· 左側 DLZ 條（射程區間）
· 目標符號（若有雷達追蹤）
· MRM（Medium Range Missile） 字樣
· 目標距離和 Vc
確認 DLZ 條出現才能正確射擊。
✅
🎯
PHASE 03

TWS 單目標射擊

🟠
TWS 射擊流程 TWS 是 AIM-120 最常用的射擊模式。雷達追蹤目標同時繼續搜索，目標 RWR 不一定有警告（比 STT 隱蔽）。射擊後執行 Crank（轉向） 保持資料鏈但降低威脅，等待 Pitbull 後脫離。
完成進度
0%
1
雷達 TWS → 建立目標追蹤，指定 DT1
雷達切換 TWS 模式，等待目標建立圓圈追蹤（2–3 次掃描）。TDC 移至目標按壓指定為 DT1。確認 DDI 右側出現 DT1 資訊：距離、Vc、高度。
✅
2
HUD 確認 DLZ 進入射程（Rne 以內）
觀察 HUD 左側 DLZ 條：
· 游標在綠色區（Rmax 附近）→ 可以射，但目標可能逃脫
· 游標進入黃色（Rne）→ 最佳射擊時機！
· 游標在紅色（Rmin）→ 太近，等距離拉開
同時確認目標 TA 約 0°（Hot，迎頭）效果最好。
✅
3
按下 Weapons Release → 「FOX 3！」
按下 Weapons Release Button（WRB，通常是搖桿扳機或 pickle button），AIM-120 發射。無線電呼號 「FOX 3！」。HUD 出現飛彈計數倒數（MISSE），DDI 顯示飛彈飛行時間。
✅
4
射擊後立即 Crank（轉向 ±60°）
發射後立即轉向 ±60° Crank（左或右偏轉 60°，但保持雷達仍能追蹤目標）：
· 降低自身被目標雷達/飛彈命中的機率
· 雷達仍追蹤 DT1，持續傳送資料鏈更新給 AIM-120
· 保持 Crank 直到 Pitbull 警告（飛彈自導）
✅
5
Pitbull 後 → 可完全脫離（Notch / 爬升 / 逃脫）
DDI 或 HUD 顯示 PITBULL 或飛行時間接近 0 時，AIM-120 已啟動自身雷達自主追蹤。此時可：
· 完全脫離，雷達不再需要鎖定目標
· 轉向應對其他威脅
· 或繼續保持追蹤監控命中情況
✅
🎯🎯
PHASE 04

TWS 多目標同時射擊

🟠
F/A-18C TWS 多目標射擊能力 F/A-18C 在 TWS 模式下可對 DT1 和 DT2 兩個目標 分別射擊，實現同時接戰兩個目標。這是 BVR 作戰最強大的能力之一，讓對手的僚機無法在友機被擊中後反應。需要 AIM-120 數量足夠（至少 2 枚）。
完成進度
0%
1
TWS 建立追蹤 → 指定 DT1 和 DT2
TWS 模式建立多個目標圓圈後：
· TDC 移至第一目標按壓 → DT1（實心圓）
· Castle Switch 短按 → 切換到第二目標
· TDC 移至第二目標按壓 → DT2（空心圓+線）
DDI 右側顯示 DT1 和 DT2 資訊
✅
2
確認兩個目標都在 DLZ 範圍
多目標射擊前確認：
· DT1 進入 Rne（黃色）
· DT2 也在射程內（距離相近最佳）
若兩目標距離差異過大，先射較近的 DT1，等 DT2 進入射程再射。
✅
3
第一次按 WRB → 射擊 DT1，「FOX 3！」
按下 Weapons Release Button，AIM-120 射向 DT1。立即執行 Crank 同時保持 TWS 追蹤。HUD 顯示第一枚飛彈計時。
✅
4
切換 DT2 → 再次按 WRB，「FOX 3 兩！」
Castle Switch 切換到 DT2 作為主要目標，確認 DT2 在 DLZ 範圍後，再次按 WRB 射出第二枚 AIM-120 飛向 DT2。兩枚飛彈同時在空中，兩個目標同時被接戰！
✅
5
射後管理：Crank 保持兩枚飛彈資料鏈
兩枚都射出後，保持 Crank 姿態，TWS 繼續追蹤兩個目標並傳送資料鏈更新給兩枚 AIM-120。等待兩枚都 Pitbull 後再考慮完全脫離。Pitbull 後才是真正的 Fire and Forget！
✅
✈️
PHASE 05

射後戰術

🟠
射後最重要：不要直飛！ 射出 AIM-120 後繼續直飛是最危險的行為，因為你也在對方的射程內！射擊 → Crank → 等 Pitbull → 判斷是否脫離是標準流程。同時注意自己的 RWR 是否有威脅警告。
完成進度
0%
1
Crank 戰術詳解
Crank 是射後最重要的機動：
· 轉向 ±60°（左轉或右轉，選擇遠離其他威脅的方向）
· 保持 Crank 角度讓雷達仍對準目標（ATA ±60° 以內）
· 同時增加距離，降低對方飛彈的能量
· 若 RWR 有警告，考慮立即 Notch + 箔條
✅
2
Pitbull 後的選項
飛彈 Pitbull（自導）後，根據情況選擇：
· 繼續支援：保持 TWS 追蹤，若飛彈丟失可再射
· 轉向脫離：爬升或轉向遠離威脅，處理其他目標
· Notch + 降低高度：若自己被鎖定，立即防禦動作
判斷依據：RWR 上是否有來襲飛彈警告
✅
3
射擊後監控命中情況
DDI TWS 頁面：
· 目標圓圈 消失 = 可能命中
· 目標圓圈 仍在且繼續機動 = 未命中，評估再射
· 注意：TWS 可能短暫丟失追蹤，不代表命中！
若未命中且仍在射程，可立即補射第二枚（若有）
✅
4
完整 BVR 接戰流程整理
距離動作
> 60NMRWS 搜索，建立態勢感知
40–60NM切 TWS，建立追蹤，指定 DT1
Rne 進入射擊 AIM-120，FOX 3！
射後立即Crank ±60°，保持資料鏈
Pitbull評估脫離或繼續支援
10NM 內準備近距纏鬥（切 AIM-9）
✅

PHASE 06

防禦來襲 AIM-120

🔴
AIM-120 來襲！如何求生？ RWR 警報 + 飛彈來襲時，需要立即且正確的防禦動作。AIM-120 是主動雷達導引，箔條有效但需配合 Notch 機動（讓都卜勒效應消失）才能有效欺騙。單靠箔條不夠！
完成進度
0%
1
RWR 警報 → 判斷威脅方向
RWR（AN/ALR-67）警報時：
· 查看 RWR 上威脅方向（點的位置）
· AWG-9 / APG-73 / AIM-120 特定警告音 = 被雷達或飛彈追蹤
· 「米格音」或固定音 = 被雷達照射（Spike）
· 快速變化音調 = 飛彈來襲！立即反應！
✅
2
Notch 機動：轉向讓威脅在正側方（90°）
Notch 是對抗都卜勒雷達飛彈最有效的機動：
· 轉向讓 來襲飛彈（或其母機雷達）在你的正側方（90°）
· 此時你對飛彈雷達的都卜勒頻移接近 0
· 飛彈難以維持追蹤
· 同時投放箔條效果加倍！
✅
3
Notch 同時投放箔條（Chaff）
執行 Notch 的同時按下 Countermeasure Dispense（CMS），投放 箔條（Chaff）。箔條製造雷達假目標，配合 Notch 機動的都卜勒效應消失，大幅提高飛彈丟失追蹤的機率。節奏：Notch → Chaff → Notch → Chaff
✅
4
低空拉近地面（地雜波掩護）
另一個有效手段：降低高度至 500–1,000 ft AGL，利用地面雜波（Ground Clutter）遮蔽飛彈雷達訊號。飛彈雷達在低空地面背景下難以分辨目標。注意：低空機動危險，需要足夠的空域！
✅
5
防禦 AIM-120 決策樹
情況動作
Spike（被雷達照）Crank，增加距離，不急
飛彈警告（遠）Notch + Chaff，Crank
飛彈警告（近）硬轉 90° Notch + 連續 Chaff
目視見到飛彈最後機動：等飛彈接近再急轉
記住：Notch 比 Chaff 更重要！方向對了箔條才有效。
✅
