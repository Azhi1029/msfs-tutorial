GBU-12 雷射導引炸彈

Paveway II · LITENING 莢艙 · CCRP / CCIP 投彈

💣
PHASE 01

GBU-12 基礎概念

🟠
GBU-12 Paveway II 簡介 GBU-12 是 500 磅雷射導引炸彈（Mk-82 + Paveway II 導引套件）。需要 雷射指定器照射目標（LITENING 莢艙），炸彈尾翼上的 雷射尋的器偵測反射雷射，導引炸彈精確命中。精度極高（CEP < 5m），是對付固定目標和裝甲的利器。
完成進度
0%
1
GBU-12 工作原理
· 炸彈本體是 Mk-82 500lb 通用炸彈
· 加裝 Paveway II 導引套件（尾翼 + 尋的器）
· 投放後依慣性彈道飛行，尋的器搜索 雷射反射光斑
· 偵測到反射光後，控制尾翼修正彈道直擊目標
· 必須全程雷射照射直到命中！
✅
2
兩種投彈模式：CCRP vs CCIP
模式全名特點
CCRPContinuously Computed Release Point水平飛行投彈，電腦計算投放點
CCIPContinuously Computed Impact Point俯衝投彈，HUD 顯示落點游標
CCRP 較安全（高空水平投彈），CCIP 較準但需俯衝暴露
✅
3
LITENING 莢艙簡介
LITENING AT 是 F/A-18C 的目標標定莢艙（Targeting Pod）：
· FLIR（前視紅外線）：夜視/熱影像鏡頭
· 內建雷射指定器：照射目標供 GBU 導引
· 可放大、追蹤、自動鎖定移動目標
· 掛載於機身下方，DDI 顯示 FLIR 影像
✅
📷
PHASE 02

LITENING 莢艙操作

🟠
SOI（Sensor of Interest）概念 DDI 顯示 FLIR 影像時，該 DDI 成為 SOI（目前控制的感測器）。TDC 游標控制 SOI 上的感測器：在 FLIR 頁面移動 TDC = 移動莢艙瞄準點。操作前務必確認 SOI 在正確的 DDI 上！
完成進度
0%
1
DDI → MENU → FLIR 開啟莢艙
選一個 DDI 按 MENU → FLIR，開啟 LITENING 莢艙影像。莢艙啟動需數秒暖機。畫面顯示熱影像（白熱或黑熱模式可切換）。
✅
2
確認 SOI 在 FLIR DDI 上
按 SOI 切換鈕（DMS, Display Management Switch） 將 SOI 切換到 FLIR 顯示的 DDI 上。畫面邊框會 高亮顯示 表示為 SOI。TDC 此時控制莢艙視野方向。
✅
3
TDC 移動莢艙視野，尋找目標
用 TDC 移動莢艙瞄準點掃視地面：
· 莢艙視野跟隨 TDC 移動
· 可調整 FOV（視野範圍）：WFOV（廣角）找目標，NFOV（窄角）放大細節
· 找到目標後 TDC 對準目標中心
✅
4
TDC 按壓 → SPI（鎖點）設定在目標
TDC 對準目標後 按壓 TDC：
· 莢艙進入 Point Track（點追蹤）模式，鎖定該地面點
· 該點成為 SPI（Sensor Point of Interest）
· 莢艙自動追蹤目標（即使飛機移動，影像也會跟隨）
· HUD 和 HSI 上也會顯示 SPI 標記
✅
5
FLIR 影像模式切換
DDI OSB 可切換：
· WHOT（White Hot）：熱源顯示白色，一般用
· BHOT（Black Hot）：熱源顯示黑色，某些情況更清晰
· Polarity（極性）切換依場景選擇
· 縮放（Zoom）調整放大倍率方便精確瞄準
✅
⚙️
PHASE 03

武器與雷射設定

⚠️
雷射代碼必須一致！ LITENING 莢艙的雷射發射代碼必須與 GBU-12 的雷射接收代碼一致（通常都設為 1688），否則炸彈尋的器無法識別反射訊號！
完成進度
0%
1
MASTER ARM → ARM
左主控台 MASTER ARM → ARM。HUD 右上角顯示 ARM。訓練時保持 SAFE！
✅
2
DDI → SMS → 選擇 GBU-12
右 DDI → MENU → SMS，確認 GBU-12 掛載。用 WSS 選擇 GBU-12 站位，HUD 左下顯示 GBU12。選擇投彈模式：CCRP 或 CCIP。
✅
3
設定雷射代碼（莢艙 & 炸彈一致）
透過 DDI 或 UFC 確認雷射代碼設定：
· 預設代碼通常為 1688（DCS 預設）
· LITENING 莢艙頁面設定 LASER CODE
· SMS 武器頁面確認 GBU-12 的 雷射接收代碼 相同
· 多人合作時需與隊友協調使用不同代碼
✅
4
確認 SPI 來源 = LITENING 莢艙
SMS 或 HUD 確認武器瞄準資料來源（SPI Source）設為 TGP（LITENING 莢艙），這樣炸彈會投向莢艙鎖定的 SPI 點，而不是其他來源（如 INS 航路點）。
✅
📐
PHASE 04

CCRP 投彈（水平投彈）

🟠
CCRP — 最安全的投彈方式 CCRP 讓你在 中高空水平飛行即可投彈，飛機不需要俯衝暴露在防空火力下。電腦持續計算投放點，飛越該點時自動或手動投放，炸彈飛行軌跡由雷射導引修正到 SPI 目標。
完成進度
0%
1
LITENING 鎖定目標 → SPI 設定完成
參考 Phase 02：用 LITENING 莢艙找到目標，TDC 按壓進入 Point Track，SPI 設定在目標上。HUD 顯示 SPI 標記和目標距離。
✅
2
確認武器模式為 CCRP
SMS 頁面確認投彈模式為 CCRP。HUD 顯示 CCRP 相關符號：
· Release Cue（投放游標）
· 飛行路徑指示
· 距離 SPI 的資訊
✅
3
飛向目標，保持航向對準 SPI
保持飛機航向讓 HUD 上的 飛行路徑標誌（FPM）朝向目標方向。維持穩定高度和速度（建議 15,000–20,000 ft，350–450 kt），讓電腦準確計算彈道。
✅
4
Release Cue 對齊 → 按住/按下 Weapon Release
當 HUD 的 Release Cue（投放標記） 與目標標記重合時：
· 自動模式 · 手動模式：按下並 持續按住 Weapon Release 直到投放完成
· 投放後 HUD 顯示 飛行時間倒數
✅
5
投放後 → 雷射持續照射直到命中
炸彈投放後，必須繼續用 LITENING 照射目標（雷射持續開啟），直到炸彈命中：
· 保持 SPI 在目標上（莢艙自動追蹤）
· 確認雷射發射開啟（LASE 鈕）
· 倒數時間結束 = 命中（DDI 可看到爆炸閃光）
✅
6
CCRP 注意事項
· 飛行高度越高，落彈時間越長，需保持雷射照射的時間越久
· 若投彈後立即轉向脫離，雷射會失去目標 → 炸彈失導
· 至少在落地前 3–5 秒 保持照射
· 多彈連投時，每枚需獨立的照射時段（或依序處理）
✅
🎯
PHASE 05

CCIP 投彈（俯衝投彈）

⚠️
CCIP — 更準但風險更高 CCIP 需要將機頭對準目標方向俯衝，HUD 顯示即時落點游標（Impact Point Cue），游標對準目標即可投彈。精度高但飛機暴露於地面防空火力時間較長，適合對付高威脅環境的小型/移動目標。
完成進度
0%
1
切換武器模式為 CCIP
SMS 頁面切換投彈模式為 CCIP。HUD 出現 Impact Point Cue（落點游標，通常為圓圈或菱形），即時顯示「若現在投彈，炸彈會落在哪裡」。
✅
2
建立俯衝航線對準目標
建立穩定俯衝：
· 俯衝角依目標類型 20–45°（越陡越準但風險越高）
· 速度建議 400–500 kt
· 機頭朝向目標方向，HUD 落點游標出現在畫面中
✅
3
落點游標對準目標 → 投彈
調整飛機姿態讓 落點游標（Impact Cue）精確對準目標，按下 Weapon Release 投彈。CCIP 不需要事先 SPI 鎖定，落點游標本身就是瞄準依據！但建議仍用 LITENING 維持照射。
✅
4
投放後立即拉起脫離（Pull Out）
投彈後 立即拉桿脫離俯衝（Pull Out），避免：
· 撞地（CFIT，最致命風險！）
· 被己方炸彈破片擊中（過低投放）
· 暴露在敵方防空火力過久
設定「最低投放高度」（Hard Deck）並嚴格遵守！
✅
5
拉起後嘗試維持雷射照射
拉起脫離時若莢艙仍能維持對目標的視線，繼續照射直到命中。若無法維持（機身遮蔽），GBU-12 仍會依慣性彈道飛向最後已知位置，精度會降低但通常仍能命中固定目標。
✅
6
CCRP vs CCIP 選擇指南
情況建議模式
高威脅防空區（SAM）CCRP（高空安全）
固定大型目標CCRP
小型/移動目標CCIP（精度優先）
低威脅環境CCIP（更直觀）
✅
