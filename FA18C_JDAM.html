PHASE 01

JDAM 基礎概念

🟢
JDAM（Joint Direct Attack Munition）簡介 JDAM 是 GPS/INS 導引套件，加裝在 Mk-80 系列炸彈尾部，使其成為 全天候精準導引炸彈。與雷射導引炸彈（GBU-12）不同，JDAM 不需要雷射照射，只需輸入目標 GPS 座標，投放後炸彈依 GPS 導航自主飛向座標點，雲層、煙霧、夜間皆不影響精度！

0%
1
JDAM 各型號（DCS 中）
型號炸彈本體重量
GBU-38Mk-82500 lb

GBU-31Mk-842,000 lb
F/A-18C 常掛載 GBU-38，平衡威力與掛載數量
✅
2
GPS/INS 導引原理
JDAM 投放後：
· 內建 GPS 接收器持續定位
· INS（慣性導航）作為 GPS 訊號中斷時的備援
· 飛彈尾翼依導航資料修正彈道，飛向預設座標
· CEP（圓概率誤差）小於 13 公尺
✅
3
JDAM vs GBU-12 vs AGM-65 比較
武器導引方式限制
JDAMGPS/INS需事先知道座標，不適合移動目標
GBU-12雷射需持續雷射照射，受天氣影響
AGM-65影像需目視鎖定，射程較短
JDAM 最適合：已知座標的固定目標、夜間/惡劣天氣攻擊
✅

PHASE 02

武器與系統設定

⚠️
關鍵：投放前 GPS 必須對準！ JDAM 需要 GPS 訊號鎖定才能精確導引。投放前確認 GPS 已對準（通常飛行一段時間後自動完成）。MASTER ARM 必須在 ARM 位置。
完成進度
0%
1
MASTER ARM → ARM
左主控台 MASTER ARM → ARM。HUD 顯示 ARM。訓練時保持 SAFE！
✅
2
DDI → SMS → 選擇 JDAM 站位
右 DDI → MENU → SMS，用 WSS 選擇 GBU-38/32 武器站。HUD 左下角顯示對應彈種代號。確認掛載數量正確。
✅
3
確認 GPS 對準狀態
DDI 或相關頁面確認 GPS（或 EGI）狀態顯示正常（通常為 綠色/READY）。GPS 通常在起飛後一段時間自動對準完成，無需手動操作，但投彈前留意是否有 GPS 故障警告。
✅
4
投彈模式：CCRP（JDAM 標準模式）
JDAM 通常使用 CCRP（Continuously Computed Release Point）模式投放，與 GBU-12 的 CCRP 操作類似，但 JDAM 的 SPI（目標點）來源是手動輸入的 GPS 座標，而非 LITENING 莢艙鎖定。
✅
🎯
PHASE 03

目標座標輸入

🔵
目標座標的來源 任務簡報通常會提供目標座標（經緯度）。也可以用 LITENING 莢艙的 Mark Point 取得座標，或從 F10 地圖讀取座標（若允許）。輸入正確座標是 JDAM 精度的關鍵！
完成進度
0%
1
DDI → SMS → JDAM 武器頁面 → 座標輸入
SMS 頁面選中 JDAM 武器站後，進入座標輸入子頁面。畫面顯示緯度（LAT）、經度（LON）、海拔高度（ELEV）輸入欄位，初始值通常為飛機目前位置。
✅
2
用 UFC 輸入緯度（LAT）
透過 UFC 輸入目標緯度，格式：N/S + 度分（小數）
例：N25042 = 北緯 25° 04.2'
· 按 N（或 S）開頭
· 輸入數字後按 ENT 確認
✅
3
用 UFC 輸入經度（LON）
接著輸入目標經度，格式：E/W + 度分（小數）
例：E121335 = 東經 121° 33.5'
· 按 E（或 W）開頭
· 輸入數字後按 ENT 確認
✅
4
輸入目標海拔高度（ELEV）
輸入目標點的海拔高度（英尺）。可從地圖、任務簡報或機上系統估算取得。高度誤差會影響投彈精度，盡量取得準確數值。
✅
5
確認座標 → SPI 設定完成
座標輸入完成後，該座標成為這枚 JDAM 的 瞄準點（SPI）。DDI 或 HUD 會顯示目標標記和距離。每枚 JDAM 可設定不同座標，實現一次出擊多目標打擊！
✅
💥
PHASE 04

投彈程序

🟢
JDAM 投彈 — 不需要照射，投了就走！ 座標設定完成後，飛到投放範圍內，依 CCRP 模式投放。投放後 JDAM 自主導航至座標，飛機可立即脫離，這是 JDAM 相較 GBU-12 最大的優勢。
完成進度
0%
1
飛向目標方向，保持穩定高度速度
朝目標座標方向飛行，HUD 顯示目標方位和距離。建議 15,000–25,000 ft，350–450 kt 穩定飛行，讓電腦準確計算投放點。
✅
2
HUD 顯示 Release Cue，對準目標
HUD 顯示 CCRP Release Cue（投放游標）和目標標記。調整航向讓游標朝向目標方向移動，JDAM 的釋放範圍（Launch Acceptability Region, LAR）較大，比 GBU-12 更寬容。
✅
3
Release Cue 對齊 → 按下 Weapon Release
游標與目標標記重合（在 LAR 範圍內）時，按下 Weapon Release！JDAM 投放離架。
· 自動模式（若設定）：電腦自動投放
· 手動模式：按住直到投放確認
✅
4
投放後立即可脫離（無需照射）
JDAM 投放後 自主依 GPS/INS 飛向設定座標，飛機不需要繼續對準目標或照射。可立即轉向脫離、攻擊下一個目標，或處理其他威脅。
✅
5
多枚 JDAM 連續攻擊不同目標
若每枚 JDAM 已設定不同座標：
1. 第一枚投放後，WSS 切換到下一枚
2. 確認該枚的目標座標正確
3. 重新對準投放
可一次飛行打擊多個分散目標，效率極高！
✅
📋
PHASE 05

戰術應用

🟢
JDAM 的戰術價值 JDAM 讓 F/A-18C 能在高空、夜間、惡劣天氣下精確打擊固定目標，且投放後立即脫離降低風險。是 SEAD（壓制敵防空）和 OCA（攻勢制空）任務中摧毀固定設施的理想選擇。
完成進度
0%
1
適合 JDAM 的目標類型
最適合：
· 固定建築物（指揮所、彈藥庫、機堡）
· 機場設施（跑道、停機坪）
· 橋樑、基礎設施
· 已知座標的 SAM 站點（先用 ISR 取得座標）
不適合：移動目標（座標投放後就固定了）
✅
2
高空高速投放戰術（Standoff）
JDAM 的滑翔能力讓飛機可以在遠離目標的高空投放：
· 高度越高、速度越快，射程（滑翔距離）越遠
· 可在 SAM 防空範圍外投放，降低風險
· 高空高速 = 更大 LAR，更安全的投彈方式
✅
3
JDAM vs GBU-12 選擇指南
情況建議武器
已知精確座標的固定目標JDAM
夜間/惡劣天氣JDAM（不受天氣影響）
移動目標GBU-12 / AGM-65
需即時視覺確認再攻擊GBU-12（LITENING）
高威脅 SAM 區域JDAM（Standoff 投放）
✅
