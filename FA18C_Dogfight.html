PHASE 01

WVR 近距空戰概念

🟡
WVR（Within Visual Range）近距空戰 WVR 是指 目視範圍內（10 NM 以內） 的空戰。與 BVR 不同，WVR 拚的是飛機機動性和飛行員技術。F/A-18C 的近距纏鬥雷達模式：GACQ / VACQ / Dogfight，以及最終武器 M61A2 機砲。
完成進度
0%
1
能量管理（Energy Management）基本概念
纏鬥中「能量」= 速度 + 高度的組合：
· 高能量（Hi-E）：高速或高高度，可以換速度換高度
· 低能量（Lo-E）：低速低高度，機動選擇受限，非常危險
· 目標：消耗敵機能量，保持自身能量優勢
· F/A-18 是能量型戰機，最佳纏鬥速度約 300–350 kt
✅
2
角速率（Turn Rate）vs 轉彎半徑
· 角速率（°/秒）：每秒轉彎幾度，越高越能快速指向敵機
· 轉彎半徑：轉彎圓圈大小，越小越好
· 速度慢 → 小半徑但低角速率
· 速度快 → 高角速率但大半徑
· F/A-18 最佳角速率速度：約 300–320 kt（Corner Speed）
· 超過 Corner Speed 拉桿只是增加阻力，不增加角速率
✅
3
F/A-18C 近距纏鬥雷達模式總覽
模式掃描方式適用情況
GACQ水平 ±20° 掃描目標在前方水平位置
VACQ垂直 ±45° 掃描目標在上方，追尾時
WACQ廣域 ±35° 掃描不確定位置時
DFMGACQ+VACQ 整合Dogfight 模式（一鍵切換）
✅
4
HOTAS 近距纏鬥快速切換
纏鬥時雙手不能離開操縱桿和油門：
· Dogfight Switch 前推（DGFT）→ 進入 Dogfight 模式（DFM），同時選 AIM-9
· Dogfight Switch 後拉（MSL OVRD）→ 飛彈超控模式
· Weapon Select Switch→ 切換 AIM-9 / AIM-120 / GUNS
· 記住：Dogfight Switch 是第一反應！進入 WVR 立即前推！
✅
🎯
PHASE 02

GACQ — 水平截獲模式

🟡
GACQ（Gun Acquisition） GACQ 是 水平自動截獲模式，雷達在 HUD 前方水平掃描 ±20° 方位角，約 8° 仰角 範圍，自動截獲第一個目標。最適合目標在正前方水平位置時使用，是追尾接戰的主要模式。
完成進度
0%
1
切換至 GACQ 模式
從正常雷達模式切換 GACQ：
· 方法一：Weapon Select Switch 向前（WSS FWD）→ GACQ
· 方法二：DDI 雷達頁按 GACQ OSB
· DDI 右上角顯示 GACQ，HUD 顯示水平掃描框
· 雷達自動在前方 ±20° 水平掃描尋找目標
✅
2
目標進入 GACQ 掃描框 → 自動截獲 STT
將飛機對準目標方向，讓目標進入 HUD 的 GACQ 掃描框（±20° 水平範圍）。雷達偵測到目標後 自動進入 STT 鎖定，HUD 出現鎖定符號和 DLZ 條（若選 AIM-9/120）。不需要手動按 TDC！自動截獲！
✅
3
GACQ 鎖定後 → 選武器射擊
STT 鎖定後依距離選擇武器：
· 2–8 NM：AIM-9 響尾蛇（若有音調鎖定）
· 0.2–0.8 NM：M61A2 機砲（GUNS 模式）
· 目標在 HUD 掃描框內飛機對準後，射擊視窗開啟
✅
4
GACQ 使用技巧
· 追尾時（Lag Pursuit）：目標在前方，GACQ 最有效
· 飛機對準目標，稍微領前（Lead）讓目標進入掃描框
· GACQ 截獲最近的目標，多目標環境小心截獲到友機！
· 若截獲失敗，轉 VACQ 或 Dogfight 模式
✅
⬆️
PHASE 03

VACQ — 垂直截獲模式

🟡
VACQ（Vertical Acquisition） VACQ 是 垂直自動截獲模式，雷達在 HUD 正前方上下掃描 ±5° 方位角，±45° 仰角 範圍。最適合目標在你正上方或追尾高角度時使用。纏鬥中目標爬升到你上方時，立即切換 VACQ！
完成進度
0%
1
切換至 VACQ 模式
從 GACQ 或其他模式切換 VACQ：
· Weapon Select Switch 向後（WSS AFT）→ VACQ
· 或 DDI 雷達頁按 VACQ OSB
· DDI 右上角顯示 VACQ，HUD 顯示垂直掃描框（細長矩形）
✅
2
目標在上方 → 機頭對準目標方向自動截獲
目標在你正上方時：
1. 機頭拉起對準目標大致方向
2. VACQ 垂直掃描框（±45° 仰角）覆蓋目標
3. 自動 STT 鎖定，HUD 出現目標符號
VACQ 特別適合：Guns 追尾拉垂直時、目標 Egg 機動後在上方
✅
3
GACQ vs VACQ 選擇原則
目標位置建議模式
正前方水平（±20°）GACQ
正上方或高仰角VACQ
不確定位置Dogfight（DFM）
目標在 HMD 視野內Dogfight + HMD
實戰建議：進入 WVR 直接切 Dogfight Switch，DFM 模式整合兩者！
✅
🥽
PHASE 04

Dogfight 模式 & HMD 離軸射擊

🟡
Dogfight 模式（DFM）是最強近距模式 按下 Dogfight Switch 前推，一鍵進入 DFM：自動選 AIM-9X、啟動 GACQ+VACQ 整合掃描、開啟 HMD（頭盔瞄準具）離軸射擊。目標不需要在機頭前方，只要頭轉向目標，AIM-9X 就能鎖定！
完成進度
0%
1
Dogfight Switch 前推 → 進入 DFM
油門上的 Dogfight/MSL Override Switch 前推（DGFT）：
· 武器自動切換至 AIM-9X
· 雷達切換 DFM（整合 GACQ+VACQ）
· HMD 啟動，頭盔顯示目標指向資訊
· HUD 右上角顯示 DFM
進入 WVR（10NM 以內）立即前推 Dogfight Switch！
✅
2
HMD 頭盔瞄準具使用
DFM 啟動後，HMD（Helmet Mounted Display）顯示：
· 目標圓圈：AIM-9X 目前指向方向
· 鎖定環：目標被 IR 導引頭捕捉時出現
· 轉頭讓 HMD 圓圈對準目標 → AIM-9X 跟隨頭部轉動
· 聽到 持續音調（growl） = AIM-9X 鎖定目標！
✅
3
HMD 離軸射擊 — 不需要對準目標！
AIM-9X 配合 HMD 可以 離軸射擊（Off-Boresight）：
· 目標在機頭側方甚至後方 45–90° 也能鎖定
· 操作流程：
　1. Dogfight Switch 前推（DFM）
　2. 頭轉向目標方向
　3. 等待 AIM-9X 音調鎖定
　4. 按 Weapon Release 射擊，「FOX 2！」
✅
4
DFM 模式雷達自動截獲
DFM 模式下，雷達同時執行 GACQ 和 VACQ 掃描：
· 目標在前方水平 → GACQ 自動截獲 STT
· 目標在上方 → VACQ 自動截獲 STT
· STT 鎖定後可射 AIM-9X（HMD）或 AIM-120
· 或切換 GUNS 用機砲攻擊
DFM 是 WVR 全能模式，纏鬥中保持在 DFM！
✅
💥
PHASE 05

M61A2 Vulcan 機砲

🔴
M61A2 Vulcan 20mm 六管機砲 F/A-18C 配備 M61A2 Vulcan 20mm 六管旋轉機砲，射速 6,000 發/分鐘，備彈 578 發（約 5–6 秒連射）。機砲是近距最後手段，必須接近到 0.2–0.5 NM！射擊時間要精準，珍惜每一發！
完成進度
0%
1
切換至 GUNS 模式
切換機砲模式：
· Weapon Select Switch 向右（WSS RIGHT）→ GUNS
· 或 DFM 模式下 WSS 切換到 GUNS
· HUD 顯示 GUNS 字樣和機砲瞄準具
· 確認 MASTER ARM → ARM！
✅
2
HUD 機砲瞄準具判讀（EEGS）
GUNS 模式 HUD 顯示 EEGS（Enhanced Envelope Gun Sight）：
· 中央十字準心：機砲射擊方向
· 漏斗線（Funnel）：依目標翼展自動計算提前量
· STT 鎖定時出現 LCOS（Lead Computing Optical Sight）：自動計算提前量，將目標飛進 LCOS 圓心射擊！
· HUD 左側顯示剩餘彈量
✅
3
機砲射擊距離與時機
· 最佳射擊距離：0.2–0.4 NM（約 400–750m）
· 太遠（0.5NM+）彈道散布大，命中率低
· 太近（0.1NM 以內）碎片可能擊中自機！
· 射擊時間：1–2 秒短促點射，不要長按浪費子彈
· 射擊後立即脫離，避免碎片
✅
4
Lag Pursuit（滯後追蹤）射擊技巧
機砲追尾的標準技術：
· Lag Pursuit：飛機位於目標轉彎弧的內側，機頭指向目標後方
· 優點：能維持距離不過衝，逐漸縮短射擊距離
· 當 LCOS 圓心對準目標時 → 射擊 1–2 秒
· STT 鎖定 + LCOS 是最有效的機砲射擊方式！
✅

PHASE 06

脫離戰術 Disengagement

🟢
知道何時脫離比知道如何纏鬥更重要！ 纏鬥最危險的錯誤是過度執著。能量耗盡、彈藥不足、多對一不利時，立即脫離才是正確選擇。F/A-18 的 GE F404 引擎推力強勁，善用加速脫離是優勢。
完成進度
0%
1
Break Turn — 緊急防禦轉彎
有飛彈或炮火來襲時的緊急機動：
· 向威脅方向做 最大 G 轉彎（6–7.5G）
· 同時投放 箔條（Chaff） 和 熱焰彈（Flare）
· 轉彎讓威脅通過身後
· Break 後立即加速重建能量！
· Break 方向：朝威脅方向，不是逃離方向！
✅
2
Extend — 加速脫離拉距離
纏鬥形勢不利時的脫離機動：
· 油門推至 MIL 或 AB（後燃器）
· 機頭指向遠離敵機方向，俯衝加速
· 目標是快速增加 1–2 NM 以上的距離
· 拉開距離後可重新設定交戰條件（再射 AIM-120）
· Extend 後若敵機追來 → 轉向 BVR 射程再射！
✅
3
脫離時機判斷
以下情況應立即考慮脫離：
· 能量過低（速度低於 200 kt，高度不足）
· 彈藥耗盡（機砲子彈少於 100 發）
· 多對一劣勢（被多架敵機圍攻）
· RWR 多重威脅警告
· 油量不足（低於返航所需）
✅
4
